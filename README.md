## Introdução

Este relatório apresenta o processo de implementação de serviços em nuvem na empresa **Abstergo Industries**, uma empresa farmacêutica fictícia que atualmente não utiliza soluções em cloud computing e deseja atuar como **revendedora para farmácias**.

O objetivo do projeto é implementar **3 serviços da AWS** com foco principal na **redução de custos operacionais**, aumento da escalabilidade e melhoria da eficiência dos processos de TI, eliminando a necessidade de infraestrutura física própria (on-premises).

---

## Descrição do Projeto

O projeto de implementação foi dividido em **3 etapas**, cada uma utilizando um serviço específico da AWS, escolhido estrategicamente para gerar **economia imediata** e **ganhos operacionais**.

### Etapa 1:
- **Ferramenta:** Amazon EC2  
- **Foco da ferramenta:** Computação em nuvem sob demanda  
- **Descrição do caso de uso:**  

Atualmente, a empresa mantém servidores físicos para hospedar sistemas internos (ERP, controle de estoque e pedidos). Com o **Amazon EC2**, esses servidores passam a ser virtuais, eliminando custos com compra de hardware, manutenção, energia elétrica e refrigeração.

**Redução de custos:**  
- Pagamento apenas pelo tempo de uso das instâncias  
- Possibilidade de usar instâncias sob demanda ou reservadas  
- Eliminação de custos com servidores físicos e manutenção  

**Ganho para a empresa:**  
- Escalabilidade conforme o crescimento do número de farmácias atendidas  
- Maior disponibilidade dos sistemas  
- Redução de falhas físicas e tempo de indisponibilidade  

---

### Etapa 2:
- **Ferramenta:** Amazon S3  
- **Foco da ferramenta:** Armazenamento de dados seguro e escalável  
- **Descrição do caso de uso:**  

A empresa armazena grandes volumes de documentos, como notas fiscais, relatórios regulatórios (ANVISA), contratos com farmácias e backups de sistemas. O **Amazon S3** substitui servidores de arquivos locais.

**Redução de custos:**  
- Armazenamento de baixo custo comparado a storage físico  
- Eliminação de gastos com servidores de backup  
- Uso de classes de armazenamento mais baratas para dados antigos  

**Ganho para a empresa:**  
- Alta durabilidade e segurança dos dados  
- Facilidade de acesso e compartilhamento entre setores  
- Backup automático e recuperação rápida em caso de falhas  

---

### Etapa 3:
- **Ferramenta:** AWS RDS  
- **Foco da ferramenta:** Banco de dados gerenciado  
- **Descrição do caso de uso:**  

Os bancos de dados utilizados para pedidos, clientes e controle de estoque deixam de ser mantidos localmente e passam a utilizar o **Amazon RDS**, que automatiza tarefas administrativas.

**Redução de custos:**  
- Menor necessidade de equipe dedicada à administração de banco  
- Redução de falhas e retrabalho  
- Backups e atualizações automáticas sem custo operacional adicional  

**Ganho para a empresa:**  
- Maior confiabilidade dos dados  
- Melhor performance dos sistemas  
- Facilidade de crescimento conforme o aumento de clientes  

---

## Conclusão

A implementação dos serviços **Amazon EC2, Amazon S3 e AWS RDS** na empresa *Abstergo Industries* tem como resultado esperado a **redução significativa de custos operacionais**, maior escalabilidade e melhoria na disponibilidade dos sistemas.

Com a migração para a nuvem, a empresa elimina gastos com infraestrutura física, paga apenas pelo que utiliza e se prepara para crescer como **revendedora farmacêutica**, atendendo mais clientes com segurança e eficiência.
